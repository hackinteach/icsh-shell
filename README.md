# README #

ICSH -- C implementation of a simple shell.

Built-in commands:
- echo <args> -- printing <args> to screen.

- jobs -- listing all jobs in the current shell.

- fg <pid> -- move job <pid> to foreground

- bg <pid> -- move job <pid> to background

- exit -- exit the shell
